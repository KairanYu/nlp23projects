## 基于字和词的中文拼写纠错实验（CSC）

本实验分别基于字和基于词实现了两种中文拼写纠错方法。

项目目录如下图所示

```
.
├── README.md
└── src
    ├── DIC # 基于词的概率中文拼写纠错
    │   ├── code # 代码实现
    │   ├── dataset # 数据集
    │   └── README.md # 运行方式
    ├── MLM # 基于字的深度学习中文拼写纠错
    │   ├── code # 代码实现
    │   ├── dataset # 数据集
    │   ├── output # 模型输出
    │   └── README.md # 运行方式
    └── report.pdf # 实验报告
```

整体实验部署思路见 report.pdf

代码实现以及运行方法分别见分文件夹下的README.md
